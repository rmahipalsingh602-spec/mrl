Onboarding

Go from first install to first output in minutes.

The official download flow is intentionally simple: install MRL, add it to your PATH, open a file, and run it with one command.

01
Download the official Windows setup

Grab the branded MRL installer from the download center and let setup place the runtime cleanly.

02
Add MRL to your PATH

Once it is available globally, the same mrl command works from any project directory.

03
Run your first file

Create a new .mrl file and launch it with mrl your-file.mrl.
