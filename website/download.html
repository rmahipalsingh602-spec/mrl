Install

Four steps to your first run.

  1. Download and launch the Windows setup installer.
  2. Complete setup and enable the PATH option so the mrl command is available globally.
  3. Open a new terminal and verify the setup with mrl --version.
  4. Run your file with mrl test.mrl or mrl page.hi.
If you use the portable EXE instead of setup, manually add its folder to PATH. Use the file name after the command: mrl test.mrl, not just .mrl.
mrl --version
mrl test.mrl

That is the public developer command surface: first verify the install, then run the file by name.
